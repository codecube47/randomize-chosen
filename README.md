
# randomize-chosen
Randomize selector for chosen.js plugin

###How to use randomize-chosen.js

     $("#selector").randomize({
        item_count:2
     })

### Full example

    $("#selector").randomize({
        item_count:2
     })

# chosen

 Chosen is a library for making long, unwieldy select boxes more user friendly.

- Prototype support: 1.7+
